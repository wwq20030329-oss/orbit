# Orbit Content Creator Brief

## Key Features

Orbit is an open-source remote control app that lets developers control AI coding agents (Claude Code, Codex, Gemini CLI) from their phone.

- Control Claude Code from iOS, Android, or web browser
- Runs on YOUR hardware (laptop, desktop, server, Raspberry Pi)
- No subscription fees or cloud costs - completely free (you still need to pay your AI model provider for Claude Code, Codex, Text to Speech, Speach to Text, etc..)
- Push notifications when agents need input or hit errors
- End-to-end encrypted connection between devices
- Run multiple coding agents in parallel
- Voice input for hands-free control
- Works with your existing dev setup and tools

## Key Value Props

- On Orbit, I can manage Claude Code from my phone without paying for a cloud relay
- On Orbit, I never miss when Claude gets stuck on a permission prompt
- Orbit runs on my own computer, not expensive cloud servers
- On Orbit, I can check on long-running tasks without returning to my desk
- Orbit is open source - I can audit, modify, or self-host everything
- On Orbit, I can run multiple agents in parallel and switch between them instantly
- Orbit keeps my code on my machines with end-to-end encryption
- On Orbit, I can keep coding while commuting, eating, or away from my desk
